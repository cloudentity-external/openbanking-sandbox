# openbanking-sandbox

## Build

```sh
make build
```

## Run

``` sh
make run
```

## URLs

``` sh
* acp admin: https://localhost:8443/app/default/admin
* acp demo client: https://localhost:8443/default/default/demo (user / user)
* tpp: http://localhost:8090
* consent-page: http://localhost:8090
```
